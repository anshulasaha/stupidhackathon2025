Can I get a Box? - Josh Zhong

Did you know there are free online 3D modelling tools you can use to create simple enclosures for your stupid projects?
Bring your laptop and a friend with 3D printing access to learn more!

Make a Little Game - Alanna Okun

We'll be talking about a couple of tools for making lightweight, flexible digital games / gamelike experiences quickly, like Twine, Bitsy, and Google Sheets (for real!).
No coding experience necessary.

Classroom 412

Build Artificially Unintelligent Websites with React + LLMs - Lachlan Campbell

React is the JavaScript library used by the largest companies to build their most important websites. We’re going to use the same tools to make tiny, stupid websites, made stupider with generative AI. We’ll cover the basics of setting up a website with Vite and React.js components, using React Hooks to add interactivity to the site, then using Replicate’s open source AI models to add the worst text/image generation you’ve ever seen. Come with basic JavaScript experience, leave with a website!
